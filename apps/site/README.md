# [TSA Webmaster 2022-2023](https://webmaster-tsa.vercel.app/)

This site is a [Next.JS](https://kit.svelte.dev/) app built with [Tailwind CSS](https://tailwindcss.com/) and [SCSS](https://sass-lang.com/) for CSS.

## Contributing

1. Install [Node.JS](https://nodejs.org/download/current) >= 16.14
1. Install the [pnpm](https://pnpm.io) package manager with `npm i -g pnpm`
1. Clone this repo (or use codespaces; whatever floats your boat)
1. Open this repo in some command prompt (bash, cmd, etc.).
1. Install dependencies with `pnpm install`
1. Spin up a dev server with `pnpm dev`
1. The dev server will start running on `http://localhost:3000`
1. Our recommended editor setup is [VSCode](https://code.visualstudio.com/) with the [Svelte](https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode) and [TailwindCSS](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) extensions.
1. Pushing to the Github repo will automatically update the [live site](https://webmaster-tsa.vercel.app/)
